# 批量文书生成工具 v1

本地 Python 桌面工具，用于律师批量生成起诉状等法律文书。

## 功能

- 📋 粘贴或导入当事人名单，自动解析
- 📄 选择 Word 模板（支持 .docx / .doc / .wps / .txt）
- 🔢 支持案号补录，二次生成含案号版文书
- 📁 自由选择输出目录，文件名自动带姓名/案号
- ✅ 自动生成 Excel 总表 + 异常报告
- 🖥️ 桌面 GUI，无需命令行

## 快速开始

```bash
pip install python-docx openpyxl pandas
python run_gui.py
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `run_gui.py` | 桌面版主程序（推荐） |
| `run_case_tool.py` | 命令行版 |
| `input_list.txt` | 名单示例 |
| `make_template.py` | 生成测试模板 |
| `requirements.txt` | 依赖清单 |

## 名单格式

```
姓名：张三
证件号：350123199901011234
金额：12800
手机号：13800000000
地址：福建省福州市鼓楼区XX路
备注：无

姓名：李四
...
```

## 模板占位符

模板中用全角括号占位，例如：

```
原告：（姓名），身份证号：（证件号），联系电话：（手机号）。
诉讼请求：返还款项（金额）元。
（备注）
```

支持占位符：`（姓名）` `（证件号）` `（金额）` `（手机号）` `（地址）` `（备注）` `（案号）`

## 输出文件命名

- 无案号：`张三_起诉状.docx`
- 有案号：`张三_（2026闽0102民初1234号）_起诉状.docx`

## 补录案号流程

1. 打开输出目录内的 `_excel/case_list.xlsx`
2. 在"案号"列填入法院受理案号
3. 重新打开工具，在区域③导入该 Excel
4. 点击生成，自动输出含案号版文书
